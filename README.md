# N8N SEO Workflows

This repository contains a collection of n8n workflows designed for SEO automation, AI marketing, content creation, and related tasks. These workflows leverage various APIs and tools to streamline processes such as blog post generation, keyword research, social media posting, and more.

## Workflows Included

### SEO and Blog Automation
- `Blog_automation.json`: Automates blog post creation and publishing.
- `Blog_Post.json`: Workflow for generating blog posts.
- `SEO___Blog_Post___Keyword_Clusters___Template_UPDATED_SKOOL (1).json`: Keyword clustering for SEO.
- `SEO___Blog_Post___Trends___Template_SKOOL (1).json`: Trend analysis for blog posts.
- `seo_audit_workflow_n_8_n.json`: SEO audit automation.

### AI and Marketing
- `AI Newsletter.json`: Generates AI-powered newsletters.
- `AI_Marketing_Team.json`: AI-driven marketing team workflows.
- `Auto LinkedIn Posts.json`: Automated LinkedIn posting.
- `LinkedIn_Post.json`: LinkedIn post creation.
- `YouTube_Strategist.json`: YouTube content strategy.

### Data Scraping and Research
- `Product_Scraper.json` and `Product_Scraper (1).json`: Product data scraping.
- `YouTube_Data_Scraper.json`: Scrapes YouTube data.
- `redacted-linkedin-scraper.json`: LinkedIn data scraping (redacted).
- `Search_Images.json`: Image search workflows.

### AI Agents and Advanced Workflows
- `Complete_AI_SDR_Setup(2).json`: AI Sales Development Representative setup.
- `Dynamic_Models.json`: Dynamic AI model workflows.
- `RAG_Reranking.json`: Retrieval-Augmented Generation with reranking.
- `Supabase_Postgres.json`: Database integration with Supabase.

### Media and Content Creation
- `Create_Image.json`: Image generation.
- `Edit_Image.json`: Image editing.
- `Faceless_Video.json`: Video creation without faces.

### Additional Folders
- `n8n/`: Contains more advanced n8n workflows, including RAG pipelines, customer support, API integrations, and agent architectures.
- `drive-download-*/`: Downloaded workflows from Google Drive, including SEO automation, calendar agents, and project management.

## Getting Started

1. **Install n8n**: Ensure you have n8n installed. You can run it locally or use the cloud version.
2. **Import Workflows**: Import the JSON files into your n8n instance.
3. **Configure Credentials**: Set up necessary API keys and credentials for services like OpenAI, Supabase, etc.
4. **Run Workflows**: Activate and test the workflows as needed.

## Prerequisites

- n8n (latest version recommended)
- API keys for integrated services (e.g., OpenAI, Perplexity, Apify)

## Usage

Each workflow is self-contained. Refer to the workflow's internal comments or n8n's interface for specific configurations.
